Comment ça marche ?

  1. Sélectionnez une activité
  2. Cliquez sur "Générer le formulaire"
  3. Un code OTP (6 chiffres) sera généré automatiquement
  4. Partagez le lien et le code OTP avec les participants
  5. Les participants devront entrer le code pour accéder au formulaire
  6. Le code expire après 3 heures
